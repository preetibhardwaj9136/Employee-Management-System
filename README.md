# Employee Management System

![Python](https://img.shields.io/badge/Python-3.10-blue) ![MySQL](https://img.shields.io/badge/MySQL-8.0-green) ![Tkinter](https://img.shields.io/badge/Tkinter-GUI-orange)

## Overview
The **Employee Management System** is a desktop application built with **Python**, **Tkinter**, and **MySQL**. It provides an interactive GUI for managing employee records with **CRUD operations**, **live search**, and **data export to CSV**. Ideal for demonstrating full-stack Python desktop app development skills.

---

## Features
- **Insert Records:** Add new employees with details like first name, last name, age, and income.  
- **Update Records:** Edit existing employee information directly from the table.  
- **Delete Records:** Remove employee records with a single click.  
- **Live Search:** Filter employees in real-time by first name, last name, or age.  
- **Export to CSV:** Save all employee data to a CSV file for backup or reporting.  
- **Professional GUI:**  
  - Striped rows with alternating colors  
  - Colored headers and buttons for intuitive navigation  
  - Responsive layout for clean presentation

---

## Technologies Used
- **Python 3** – Core programming language.  
- **Tkinter** – GUI framework for desktop applications.  
- **MySQL** – Backend database for storing employee records.  
- **CSV Module** – For exporting employee data.
